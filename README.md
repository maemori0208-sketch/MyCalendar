# MyCalendar

Googleカレンダー風のUIに、**タスク管理**と**議事録（会議メモ）作成**を統合したブラウザアプリです。
ビルド不要・依存ライブラリなしのバニラJavaScriptで動作し、データはブラウザの `localStorage` に保存されます。

## 起動方法

`index.html` をブラウザで開くだけです。

- **簡単**: `index.html` をダブルクリック
- **推奨**（ローカルサーバー経由）:
  ```bash
  cd MyCalendar
  python -m http.server 4599
  # → http://localhost:4599 をブラウザで開く
  ```

## 機能

### 📅 カレンダー
- **月 / 週 / 日** の3ビュー切替
- 日付セル・時間スロットをクリックして予定を作成
- 予定の編集・削除、終日予定、7色のカラー分け、説明メモ
- **予定のドラッグ移動**（月ビュー＝別の日へ / 週・日ビュー＝時間・日を変更）。マウス・タッチ両対応
- **繰り返し予定**（毎日 / 毎週 / 毎月 / 毎年、終了日指定）。繰り返し予定の編集・削除・移動時は「この予定のみ / すべての予定」を選択可能。↻ マークで表示
  - 「毎月」は **日付ベース（毎月◯日）** に加え、**曜日ベース（毎月 第N◯曜日 / 毎月 最終◯曜日）** を選択可能（例：毎月第3日曜日）
- **マウスホイールで期間移動**（月ビュー全体 / 週・日ビューはヘッダー上）
- **スワイプで期間移動**（スマホ）— 左スワイプで次へ、右スワイプで前へ
- 週/日ビューには現在時刻ライン、月ビューには「他 N 件」表示
- タスクの期限・議事録もカレンダー上に重ねて表示（サイドバーでON/OFF）

### ✓ タスク管理
- クイック追加（タイトル・期限・優先度）
- 完了トグル、優先度（高/中/低）、期限表示（今日/明日/日付、期限切れは赤）
- フィルタ（すべて / 未完了 / 完了 / 今日）
- 未完了件数をサイドバーにバッジ表示

### 📝 議事録（会議メモ）
- 会議名・日時・出席者・アジェンダ・議事内容を記録
- 入力は**自動保存**
- **アクションアイテム**（担当・期限つき）を管理
- アクションアイテムを **ワンクリックでタスク化** → タスク管理へ連携

### データの同期・バックアップ
- **自動同期（クラウド）** — サイドバー →「自動同期（クラウド）」から設定。3方式から選択:
  - **Google ドライブ（推奨・Firebase不要）**: Googleログイン（Google Identity Services）＋ Drive API。データは**あなたのGoogleドライブのアプリ専用フォルダ(appDataFolder)**に `mycalendar.json` として保存。データベース構築は不要。初回だけGoogle CloudのOAuthクライアントIDを用意し、以降は各端末で「Googleでログイン」するだけ。約15秒間隔のポーリング＋変更時アップロードで同期
  - **Google ログイン（Firebase）**: Firebase Authentication でGoogleログイン。`/users/{uid}` に保存、ルールで**本人のみ読み書き可**
  - **同期コード（ログイン不要）**: 両端末で同じ「データベースURL」＋「同期コード」。`/spaces/{code}` に保存
  - オフライン時はローカル保持、オンライン復帰で同期。競合はドキュメント全体の後勝ち
- **手動同期（エクスポート / インポート）** — 全データをJSONで書き出し／取り込み。エクスポートはスマホでは共有シート対応。インポートは **統合（マージ）** または **全置換** を選択可能

### Googleカレンダー取り込み（読み取り専用）
- サイドバー →「Googleカレンダー取り込み」から設定。Googleログイン（Google Identity Services）＋ Google Calendar API（`calendar.readonly`）で、**Googleカレンダーの予定をアプリに表示**します
- **あなたのGoogleカレンダーには一切書き込みません**（取り込みのみ）。予定は「G」印付きで表示され、タップするとGoogleカレンダーで開きます
- 表示するカレンダーを選択可能。取得結果はローカルにキャッシュし、オフラインでも直近の内容を表示
- サイドバーの「表示」で Googleカレンダーの表示ON/OFFを切替
- 初回のみ Google Cloud の OAuthクライアントID（Drive連携と同じIDを流用可）が必要。設定手順はモーダル内に記載
- ※ 起動時に自動でポップアップを出さない設計のため、最新へ更新するには連携画面から「Googleでログイン」（更新）をタップします

### 表示アニメーション
- 期間の前後移動は**スライド**、月/週/日の切り替えは**フェード**でアニメーション（スワイプ・矢印・ホイール・ショートカット共通）。OSの「視差効果を減らす（prefers-reduced-motion）」設定時は自動でオフ

### その他
- **ダークモード** — ヘッダー右の🌙ボタンで切替。初回はOSの設定（prefers-color-scheme）に追従し、以後は選択を記憶
- **スマホ対応** — サイドバーはハンバーガーメニューからのスライドインドロワー化。ヘッダー・タスク・議事録・モーダルもレスポンシブ。ドラッグ移動はタッチ操作にも対応
- 横断検索（予定・タスク・議事録）
- ミニカレンダーで日付ジャンプ
- キーボードショートカット（カレンダー画面）:
  - `M` 月 / `W` 週 / `D` 日 / `T` 今日 / `←` `→` 前後へ移動 / `Esc` モーダル・ダイアログを閉じる

## ファイル構成

```
MyCalendar/
├── index.html      画面レイアウト
├── styles.css      スタイル（Googleカレンダー風）
└── js/
    ├── store.js    localStorage永続化・共有ステート・日付ユーティリティ
    ├── calendar.js 月/週/日ビューの描画
    ├── tasks.js    タスク管理
    ├── notes.js    議事録
    └── app.js      初期化・ルーティング・モーダル・検索
```

初回起動時はサンプルデータが表示されます。データを空にしたい場合はブラウザの `localStorage`（キー `mycal.*`）を削除してください。
